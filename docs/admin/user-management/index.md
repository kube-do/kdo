在Kubernetes集群中，访问控制是保障集群安全的重要组成部分。其中，权限管理是访问控制的核心。
Kubernetes 的权限管理是通过其内置的 Role-Based Access Control (RBAC) 系统来实现的。
RBAC 提供了细粒度的访问控制能力，允许你定义谁可以对集群中的资源做什么操作。
