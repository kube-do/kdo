## 项目概览

{: .note }
在**项目概览**页面，可以对当前项目有个整体认识，包括基本信息，资源使用，监控信息，事件等。

![环境概览](imgs/env.gif)


## 环境概览

{: .note }
在**环境概览**页面，可以对当前项目的当前环境有个整体认识，包括基本信息，资源使用，监控信息，事件等。

![环境概览](imgs/env.gif)


## 资源搜索

{: .note }
在**资源搜索**页面，可以通过资源的名字或IP搜索对应的资源，包括:无状态应用，有状态应用，配置文件，服务，Ingress等

![环境概览](imgs/search.gif)
